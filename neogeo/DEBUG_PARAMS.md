# DOOM-NG VSLICE debug params

Generated by the param-audit workflow (read + verified vs neogeo/main.c). In debug (SPACE), WASD grid-nav, N/M adjust, P toggles BSP-walk viz. This doc is a SNAPSHOT; `neogeo/main.c` (the `vs_defaults()` boot/reset source + the param-adjust `case` switch) is the source of truth if anything has drifted.

Cost legend: **WALK**=BSP node-box tests (bx) · **EMIT**=SCB strip writes (em) · **FLATS**=vs_lut (fl) · **PROJ**=per-seg · **none**=cosmetic.
